# 🍞 Tre Fratelli — Paninoteca Romana

Projeto didático de desenvolvimento web utilizando:

- HTML5
- CSS3
- Bootstrap 5
- PHP
- MySQL
- Git/GitHub

O projeto simula o desenvolvimento de um sistema web real para uma paninoteca fictícia localizada em Roma.

---

# 🎯 Objetivos do Projeto

O sistema será utilizado inicialmente para:

- divulgação da paninoteca;
- apresentação do cardápio;
- controle interno de pedidos;
- controle de caixa;
- gerenciamento administrativo básico.

Futuramente:

- delivery;
- autenticação de usuários;
- dashboard administrativo;
- relatórios;
- controle de estoque.
