## 📖 Visão Geral do Projeto

Tre Fratelli — Paninoteca Romana é um projeto didático criado para ensino médio técnico.

Objetivos:

- ensinar desenvolvimento web;
- apresentar organização profissional;
- ensinar Git/GitHub;
- trabalhar responsividade;
- desenvolver projetos colaborativos.

O projeto simula um sistema real de uma paninoteca italiana.
