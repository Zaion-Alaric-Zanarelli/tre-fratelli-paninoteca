# 🔖 Controle de Versões

O projeto utilizará:

## Beta
Versão em desenvolvimento.

Exemplo:

```text
v0.1.0-beta
```

---

## Release Candidate

Versão quase pronta.

```text
v1.0.0-rc.1
```

---

## Stable Release

Versão estável.

```text
v1.0.0
```

---

# 🌳 Branches

| Branch | Objetivo |
|---|---|
| main | versão estável |
| beta | desenvolvimento |
